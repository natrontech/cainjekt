You must provide a CA bundle for injection to work. Either:

  1. Set caBundle in values:
     helm upgrade {{ .Release.Name }} natrontech/cainjekt \
       --set-file caBundle=/path/to/ca-bundle.pem

  2. Create a ConfigMap manually and reference it:
     kubectl create configmap my-ca-bundle \
       --from-file=ca-bundle.pem=/path/to/ca-bundle.pem \
       -n {{ include "cainjekt.namespace" . }}

     helm upgrade {{ .Release.Name }} natrontech/cainjekt \
       --set caBundleExistingConfigMap=my-ca-bundle
{{- end }}

    annotations:
      cainjekt.natron.io/enabled: "true"
